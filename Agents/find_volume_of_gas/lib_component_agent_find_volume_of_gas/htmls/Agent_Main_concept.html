Задачей агента поиска понятий, которые определяются на основе заданного понятия является нахождение понятий, которые определяются на основе заданного. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска понятий, которые определяются на основе заданного. Аргументом запроса является понятие, через которое определены другие понятия.
