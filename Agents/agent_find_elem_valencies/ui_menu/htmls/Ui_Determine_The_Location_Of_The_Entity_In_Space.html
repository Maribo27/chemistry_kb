Команда определения расположения сущности в пространстве предназначена для определения расположения сущности в пространстве, поданной на вход данной команде. Аргуметом запроса является компонент пользовательского интерфейса, расположение которого необходимо определить. Результатом выполнения команды является найденное расположение исходного компонента.
